# two-tier-infra-setup
Two Tier Infrastructure Setup Project
 
Provisioning 2 Tier infrastructure  
 
• A frontend/Tier 1 system running Apache Web Server 
 
• A backend/Tier 2 system running Python 3 and Java 11 
 

Key Info  
1. Each tier runs two nodes in different AZ (us-east-1a and us-east-1b) for redundancy. 
2. AMI template was created for node provisioning to expedite VM creation. 
3. All scripts used for nodes stored in a repo.
